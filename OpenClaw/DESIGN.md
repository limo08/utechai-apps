# DESIGN.md

## 气质与意象

关键词：自主干活、AI 龙虾助手、红色能量、终端交互、本地私有部署

场景叙述：深夜的终端前，一行行命令在暗色背景上跳动，红色光标闪烁——AI 龙虾正在替你执行任务。它是真正能自主干活的数字员工，不需要你时刻盯着，它会自己找资源、调接口、出报告。你只管下达指令，剩下的交给它。

## 视觉策略

- 图形语言：以"龙虾/爪"为核心符号，搭配终端代码行、红色脉冲光点
- 整体风格：深色沉浸式，红色能量感，终端风交互演示
- 避免：通用的科技蓝渐变、卡通化插图、过度装饰

## 配色方案

## 字体排版

- 中文优先：系统字体栈 `"PingFang SC", "Microsoft YaHei", sans-serif`
- 英文/代码：`"Inter", "JetBrains Mono", monospace`
- 标题大而粗（font-weight: 700, 72px），正文轻而松（font-weight: 400, line-height: 1.75）
- 排版节奏：标题区紧凑有力，功能区间距宽松呼吸感

## 动效与交互

- 下载按钮：脉动缩放动画 (scale 1→1.1)，红色 box-shadow 呼吸
- 品牌文字：渐变色位移动画 (gradientShift 6s)
- 功能卡片：hover 上浮 + 红色边框发光
- 终端演示：淡入淡出切换，打字机效果
- 标签脉冲：红色光点扩散动画
- 所有过渡：cubic-bezier(0.4, 0, 0.2, 1)，时长 200-300ms
- 页面入场：元素从下方淡入

## 页面结构

1. **导航栏**：Logo + 品牌名 + 导航链接（首页/平台集成/核心功能）+ 立即下载按钮
2. **Hero 区**：标签脉冲 + 渐变大标题"OpenClaw 中文版" + 副标题描述 + 下载按钮 + 平台支持说明 + 终端交互演示
3. **平台集成**：标题 + 模型 Logo 横向滚轮展示
4. **核心功能**：3 列网格功能卡片
5. **底部 CTA**：再次强调下载 + 下载按钮
6. **侧边固定**：右下角浮动下载引导

## 设计禁忌

- 禁止使用蓝紫色渐变作为主色
- 禁止使用卡通风格或可爱元素
- 禁止在深色背景上使用低对比度灰色文字
- 禁止使用过长的段落，保持简短有力
