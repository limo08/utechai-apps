# DESIGN.md

## 气质与意象

关键词：温暖泥土、终端深处、夜间书房

想象一个深夜的书房，窗外城市灯火微明，桌面上终端的绿色光标安静闪烁。没有炫目的霓虹，只有陶土色的暖光与代码行间的沉思。这是 Claude Code 的世界——沉稳、精密、值得信赖。

## 配色方案

- **主背景**：深炭黑 `#1a1a2e` → 午夜书房的墙壁
- **次背景**：暖灰 `#16213e` → 窗外远处微光
- **品牌强调**：赤陶橙 `#d4704e` → 陶土花瓶上的暖光
- **终端绿**：`#4ade80` → 终端光标的脉冲
- **代码紫**：`#a78bfa` → 语法高亮的柔光
- **文字主色**：奶白 `#f0ece2` → 羊皮纸上的墨迹
- **文字次色**：暖灰 `#94a3b8` → 阅读灯光下的阴影

## 字体排版

- 标题：系统无衬线字体，粗体（700），字间距略紧
- 正文：系统无衬线字体，常规（400），行高 1.75
- 代码：等宽字体（JetBrains Mono / SF Mono / Menlo）
- 排版节奏：大标题 56px → 区域标题 32px → 正文 16px → 小字 14px

## 视觉策略

- 大面积深色背景，辅以细微径向渐变制造空间深度
- 终端模拟窗口作为核心视觉元素，展示真实交互
- 代码片段用半透明卡片承载，边框微光
- 图标采用线性风格（Lucide），24px，2px 描边

## 动效与交互

- 页面滚动：区域渐入（fade-up），缓动 `cubic-bezier(0.16, 1, 0.3, 1)`
- 终端光标：1s 闪烁动画
- 按钮 hover：微上浮 + 光晕扩散
- 代码块：复制按钮 hover 显现，点击后短暂变绿确认
- 导航：sticky，滚动后背景模糊化

## 页面结构

1. **导航栏**：Logo + 锚点导航 + 下载按钮
2. **Hero**：大标题 + 副标题 + 终端模拟窗口（模拟 claude 启动动画）+ 双 CTA（npm 安装 / 桌面版下载）
3. **核心特性**：6 宫格卡片（深度代码感知 / 跨文件编辑 / Git 工作流 / 终端原生 / IDE 集成 / 200K 上下文）
4. **安装指南**：三步流程 + 各平台 Tab 切换（macOS / Linux / Windows）
5. **IDE 集成**：VS Code + JetBrains 双栏展示
6. **Footer**：Anthropic 品牌 + 链接

## 设计禁忌

- 禁止使用科技蓝/紫渐变作为主色
- 禁止卡片使用纯白底
- 禁止使用 3D 渲染图或浮夸装饰
- 禁止过度动画（不超过 2 个同时运行的动效）
- 禁止在深色背景上使用纯黑文字
